# CONTRIBUTING

### Introduction

First of all thank you for being here =)
Just remember, that all humans on a planet are Humans!

### How to make a thing?

1. Implement feature / fix the bug / etc.
2. Call your brach like `feature/<issue-id>-<short-description>` / `fix/<issue-id>-<short-description>`.
3. Write/change tests.
4. Write documentation (for feature) in `README.md` file.
5. Run linter by command `uv run flake8 --select WPS melony/`
5. Run tests by command `uv run pytest -v`
6. Make PR with description.
7. Enjoy! =)

### If i cannot make a feature?

Just remember, that you are able to create an issue (feature request or bug report), i really appriciate it. That small thing can change something.

# Good luck!
